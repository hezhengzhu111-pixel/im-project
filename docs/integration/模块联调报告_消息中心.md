1. 私聊：打开会话 → 拉取历史 → 发送文本 → 对端可见 → 会话列表更新
2. 群聊：创建群 → 发送群消息 → 成员可见 → 已读标记生效

## 异常流程（至少 5 条）

1. 未登录调用 conversations（401）
2. 发送消息参数缺失（4xx）
3. 网关返回 5xx（模拟后端故障）
4. 3G 网络下拉取历史超时（超时）
5. 网关限流（429/自定义限流码）下重试策略生效
