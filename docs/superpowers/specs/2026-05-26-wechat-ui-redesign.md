# 聊天核心页面微信风格 UI 重构

## 概述

参考微信桌面端和移动端的交互逻辑与 UI 界面，全局替换现有靛蓝/玻璃拟态设计为
微信风格扁平设计。优先重构聊天核心页面（会话列表 + 消息区 + 输入区），
桌面端改为 2 栏布局，移动端改为 4 Tab 布局。

---

## 子项目范围

本次聚焦 **聊天核心页面**，其他页面（设置、朋友圈等）在后续迭代中逐步对齐。

### 暗黑模式

- 背景 `#111111`、面板 `#191919`、气泡区 `#111111`
- 接收气泡 `#262626`、发送气泡 `#05A351`
- 文字 `#E5E5E5`、辅助文字 `#666666`

### 字体

- 全局：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif`
- 消息正文：`15px` / 行高 `1.5`
- 时间/辅助：`11px`
- 昵称：`13px`
- 会话列表名称：`15px`
- 会话列表预览：`13px`

### 圆角

- 消息气泡：`4px`
- 头像：`4px`
- 按钮/输入框：`4px`
- 全局统一缩减

### 去除项

- 所有玻璃拟态（`backdrop-filter: blur()`）
- 所有渐变背景
- `box-shadow` 减少到最小（仅输入框聚焦时保留微弱阴影）
- Element Plus 默认大圆角覆盖

## 第 2 部分：桌面端 2 栏布局

```
┌────────────────────┬──────────────────────────────────────┐
│ 会话列表 (280px)    │ 聊天区 (flex-1)                       │
│                    │                                      │
│ ┌────────────────┐ │ ┌──────────────────────────────────┐ │
│ │ ○ 当前用户      │ │ │ ○ 对方昵称  [在线绿点]      ⋯  │ │ ← Header 56px
│ └────────────────┘ │ └──────────────────────────────────┘ │
│ ┌────────────────┐ │ ├──────────────────────────────────┤ │
│ │ 🔍 搜索        │ │ │                                  │ │
│ └────────────────┘ │ │  消息列表 (flex-1, bg=#EDEDED)    │ │
│                    │ │                                  │ │
│ ┌────────────────┐ │ ├──────────────────────────────────┤ │
│ │ ○ 张三         │ │ │ 😊 📁 🎤  [输入...]      [发送] │ │ ← 输入区 ~140px
│ │   你好... 09:30│ │ └──────────────────────────────────┘ │
│ └────────────────┘ │                                      │
│ ┌────────────────┐ │                                      │
│ │ ● 李四    [红] │ │                                      │
│ │   图片   09:28 │ │                                      │
│ └────────────────┘ │                                      │
└────────────────────┴──────────────────────────────────────┘
```

### 左侧：会话列表面板 (280px)

- **顶部区域 (64px):** 当前用户头像(36px) + 用户名，无全局搜索栏
- **搜索框:** `el-input` 简化样式，placeholder "搜索"，前置搜索图标
- **会话列表项:**
  - 左侧 40px 头像 + 红点未读标记（覆盖在头像右上角）
  - 中间：昵称(15px) + 最后消息预览(13px, 单行截断, #B0B0B0)
  - 右侧：时间(11px, #B0B0B0)
  - 选中态：`#E5E5E5` 背景
  - 高度 64px，不显示 pin/mute/online/AI 等额外标记
- **滚动:** 简单 overflow-y: auto

### 右侧：聊天区 (flex-1)

- **Header (56px):**
  - 左：对方小头像(32px) + 昵称(15px) + 在线绿点(8px, #07C160)
  - 右：加密锁图标(仅加密态显示) + 更多 `⋯` 按钮(32px 图标按钮)
  - 背景 #F7F7F7，底部分割线 0.5px
- **消息列表:** flex-1, bg=#EDEDED
- **输入区 (~140px):**
  - 工具栏行：3 个图标按钮(😊 📁 🎤) + 空格
  - 文本输入：textarea，min-height 40px, max-height 120px, 无边框, bg=#F7F7F7
  - 发送按钮行：右对齐，`#07C160` 背景，白色文字，宽 68px 高 32px
  - 全体 bg=#F7F7F7，顶部分割线

### 移除的现有组件

- `SideNavBar`（64px 窄导航栏）→ 功能融入会话列表顶部
- `ConnectionStatusBar` → 状态移入 Header
- 右侧 `DetailPanel` (300px) → 功能融入 `⋯` 下拉菜单
- `ChatEncryptionBanner` → 已无用，删除
- 所有 `glassmorphism` 类

## 第 3 部分：消息气泡

### 接收消息 (对方)

```
┌──────────────────────┐
│ ○ 头像(36px) 昵称    │  ← 昵称仅群聊显示
│ ┌──────────────┐     │
│ │ 消息内容      │     │  ← 白底 #FFF, 圆角 4px, 内边距 10px 14px
│ └──────────────┘     │
│ 09:30                │  ← 11px #B0B0B0
└──────────────────────┘
```

### 发送消息 (自己)

```
┌──────────────────────┐
│              昵称 ○  │
│     ┌──────────────┐ │
│     │ 消息内容      │ │  ← 绿底 #95EC69, 圆角 4px, 内边距 10px 14px
│     └──────────────┘ │
│            09:30 ✓  │ │  ← ✓ 已读/已发送状态
└──────────────────────┘
```

### 规格

- 气泡最大宽度：60% 聊天区宽度
- 连续同人消息：头像隐藏，间距 2px，不重复显示昵称和时间
- 消息类型支持：TEXT / IMAGE / FILE / VOICE / VIDEO / SYSTEM
- 图片消息：最大 240px 宽，圆角 4px
- 语音消息：播放按钮 + 时长，气泡宽度根据时长变化
- 日期分隔：居中文字标签 `5月26日 周一`，12px #B0B0B0
- 系统消息：居中灰色文字 12px，例 "以上为未读消息"
- 加密消息（解密失败）：居中灰色文字 "加密消息暂无法解密"
- 消息状态：`⌛` 发送中、`✓` 已发送、`✓✓` 已读、`!` 发送失败

### 头像

- 个人/群组统一 36px，圆角 4px
- 默认头像使用灰色占位图

## 第 4 部分：移动端 4 Tab 布局

```
┌─────────────────────────────────┐
│  ← 张三              [绿点] ⋯ │  ← 导航栏 48px, bg=#EDEDED
├─────────────────────────────────┤
│                                 │
│   消息列表 (bg=#EDEDED)          │
│   (全屏, 有虚拟滚动)             │
│                                 │
├─────────────────────────────────┤
│  😊 📁 🎤  [输入...]     发送   │  ← 输入区
├─────────────────────────────────┤
│  💬 微信 │ 👤 通讯录 │ 🔍 发现 │ 👤 我 │  ← 底部 Tab 56px + safe-area
└─────────────────────────────────┘
```

### 4 个 Tab

| Tab | 路由 | 功能 |
|-----|------|------|
| 微信 | `/chat` | 会话列表 → 点击进入全屏聊天 |
| 通讯录 | `/contacts` | 好友列表（字母分组）+ 群组列表 |
| 发现 | `/moments` | 朋友圈入口 |
| 我 | `/profile` | 个人资料 + 设置 + 退出 |

### 交互

- 每个 Tab 独立视图栈，点进聊天时全屏推入（带返回箭头）
- 底部 Tab 在聊天全屏视图内**始终可见**（微信风格）
- 手势返回：后续迭代
- 输入区在移动端使用 `font-size: 16px`（防止 iOS 缩放）
- 安全区适配：`padding-bottom: env(safe-area-inset-bottom)`

### 与现有移动端的差异

- 现有 `MobileTabBar` 5 Tab（聊天/联系人/群组/朋友圈/设置）→ 合并为 4 Tab（微信/通讯录/发现/我）
- 群组合入"通讯录" Tab
- 设置合入"我" Tab
- 去除独立的 Groups 路由
